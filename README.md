## Literalura
Aplicación en Java que a partir de la API Gutendex permite la busqueda de un libro, permitiendo el registro de este libro, así como el autor en una base de datos, permitiendo la persistencia de datos mediante JPA. 
## ¿Cómo usar?
Basta con correr la aplicación, al momento de ejecutarla se mostrará una serie de opciones disponibles, al buscar el titulo del libro se agregará a su respectiva tabla, de igual forma al autor. A partir de estos registros en base de datos es posible hacer una serie de consultas, como observar todos los libros o autores registrados, clasificar libros por idioma o incluso conocer libros por el año en el que los autores estaban con vida.
